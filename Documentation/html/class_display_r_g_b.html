Detailed Description

Brief Display Kinect v2 RGB image on a solid.

Attach this script to a cube with aspect ratio of 16:9 (same as RGB image retrieved by the sensor) in one of their faces. Drag the object containing the SensorRGB script to the cube containing this script.
